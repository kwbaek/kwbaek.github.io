OpenClaw로 AI 에이전트 자동화를 시작하는 법 - 크론잡과 하트비트 활용

February 12, 2026

AI 비서가 24시간 일해준다면?#

OpenClaw는 단순한 AI 챗봇이 아닙니다. 자율적으로 작동하는 AI 에이전트로, 사용자가 자리를 비운 동안에도 정해진 작업을 수행할 수 있습니다. 저는 매일 밤 9시마다 AI/ML 트렌드를 수집하고, 이를 기반으로 블로그 포스팅까지 자동으로 진행하는 시스템을 구축했습니다. 어떻게 가능할까요?

크론잡(Cron Job)과 하트비트(Heartbeat)#

OpenClaw의 핵심 자동화 기능은 두 가지입니다:

1. 크론잡 - 정해진 시간에 작업 실행#

크론잡은 특정 시간이나 주기마다 AI 에이전트가 작업을 수행하도록 예약하는 기능입니다.

예를 들어 제 블로그 포스팅 크론잡은 이렇게 설정되어 있습니다:

{
  "name": "Daily Blog Post - AI/ML Trends & OpenClaw",
  "schedule": {
    "kind": "cron",
    "expr": "0 21 * * *",  // 매일 밤 9시
    "tz": "Asia/Seoul"
  },
  "payload": {
    "kind": "agentTurn",
    "message": "매일 블로그 포스팅 시간이야! ..."
  },
  "sessionTarget": "isolated"
}

이 크론잡은 매일 밤 9시 정각에 실행되어, Discord 채널에서 오늘의 AI/ML 트렌드를 읽고, 블로그 글 2개를 작성한 뒤, Git으로 자동 배포까지 완료합니다.

2. 하트비트 - 주기적 체크와 능동적 대응#

하트비트는 일정 주기마다 AI가 상황을 체크하고 필요할 때만 행동하는 기능입니다.

제 경우 HEARTBEAT.md 파일을 비워두었기 때문에 하트비트를 사용하지 않지만, 다음과 같은 용도로 활용할 수 있습니다:

# HEARTBEAT.md

## 체크 항목 (하루 2-4번 체크)

- **이메일**: 중요한 미확인 메일이 있나?
- **캘린더**: 24-48시간 내 일정이 있나?
- **날씨**: 외출이 예상되면 날씨 정보 확인
- **GitHub**: PR이나 이슈에 대응이 필요한가?

하트비트는 30분~1시간 간격으로 실행되며, 위 항목들을 체크한 뒤 문제가 있을 때만 알림을 보냅니다. 아무 일이 없으면 조용히 HEARTBEAT_OK를 반환하죠.

크론잡 vs 하트비트: 언제 뭘 써야 할까?#

크론잡이 적합한 경우 하트비트가 적합한 경우
정확한 시간에 실행돼야 함 대략적인 주기로 체크하면 됨
매번 반드시 실행돼야 함 상황에 따라 스킵 가능
독립적인 작업 (이전 대화 불필요) 최근 대화 맥락이 필요함
특정 채널에 결과 전달 메인 세션에 알림

예시:

  • ✅ 크론잡: “매일 오전 9시 업무 보고서 작성”
  • ✅ 하트비트: “이메일함 확인하고 중요한 거 있으면 알려줘”

실전 활용 사례: 제 블로그 자동화#

제 블로그 자동화 워크플로우는 이렇습니다:

  1. 크론잡 실행 (매일 밤 9시)
  2. Discord #ai-ml-trends 채널에서 오늘 수집된 트렌드 읽기
  3. 가장 중요한 트렌드 2-3개 선별
  4. AI/ML 트렌드 블로그 글 작성 (카테고리: ai-ml)
  5. OpenClaw 활용 팁 블로그 글 작성 (카테고리: openclaw)
  6. Hugo 블로그에 마크다운 파일 생성
  7. Git add, commit, push
  8. Hugo 빌드 후 GitHub Pages 배포

모든 과정이 완전 자동입니다. 제가 자는 동안 AI가 일을 끝내고, 아침에 일어나면 새 글이 블로그에 올라와 있습니다.

모델 선택: 비용 vs 품질#

크론잡에서는 어떤 모델을 사용할지 선택할 수 있습니다. 제 경우:

  • 메인 세션: Claude Opus 4.6 (최고 품질)
  • 크론잡: Claude Sonnet 4.5 (비용 절약)

트렌드 수집이나 블로그 작성은 Sonnet으로도 충분하기 때문에, 크론잡에 고가의 Opus 모델을 쓸 필요가 없습니다. 비용 대비 효율을 고려한 선택이죠.

시작하기: 첫 크론잡 만들어보기#

OpenClaw에서 크론잡을 만드는 건 간단합니다. cron 도구를 사용하면 됩니다:

openclaw cron add \
  --schedule "0 9 * * *" \
  --tz "Asia/Seoul" \
  --message "오늘의 날씨와 일정을 요약해줘" \
  --target isolated

또는 Discord/Telegram에서 직접 대화로 설정할 수도 있습니다:

“매일 오전 9시에 날씨 체크해서 알려줘”

AI가 알아서 크론잡을 설정해줍니다.

마무리: AI가 일하는 동안, 당신은 자유롭게#

OpenClaw의 크론잡과 하트비트는 AI를 진짜 비서로 만드는 핵심 기능입니다. 단순히 질문에 답하는 것을 넘어, 능동적으로 작업을 수행하고, 필요할 때 알림을 보내는 진정한 자율 에이전트가 되는 거죠.

여러분도 반복적인 작업이 있다면, OpenClaw에게 맡겨보세요. 시간이 지날수록 점점 더 많은 일을 대신해줄 것입니다.
