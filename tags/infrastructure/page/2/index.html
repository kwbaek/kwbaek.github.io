TokenArena는 같은 모델도 제공 endpoint에 따라 정확도, tail latency, joules per correct answer가 크게 달라질 수 있음을 보여줍니다. 이는 에이전트 운영에서 “GPT 계열을 쓸까, Claude 계열을 쓸까”보다 더 낮은 레이어의 질문이 중요해진다는 뜻입니다. 같은 모델명이라도 어떤 provider, 어떤 serving SKU, 어떤 batching 정책, 어떤 latency budget에서 부르느냐에 따라 비용과 품질이 갈립니다. 에이전트가 한 작업에 수십 번 도구와 모델을 호출한다면 이 차이는 곧 제품 마진과 사용자 경험 차이가 됩니다. ...

Meta의 humanoid AI 강화를 위한 robotics 스타트업 인수, NVIDIA의 DeepSeek V4 배포 가이드, Crab 같은 agent sandbox checkpoint 연구, Exploration Hacking과 FlashRT 같은 post-training·red-teaming 연구가 모두 같은 방향을 가리킵니다. 이제 중요한 질문은 “모델이 똑똑한가?”보다 “이 시스템을 오래, 싸게, 안전하게 굴릴 수 있는가?”에 가까워졌습니다. 첫 번째 축은 physical AI입니다. AI는 모델 API 시장을 넘어 전력, 칩, 클라우드, 데이터센터, 배포 파트너까지 묶인 풀스택 인프라 산업이 되고 있습니다. 특히 Google Cloud와 OpenAI의 움직임은 상징적입니다. OpenAI도 Stargate 업데이트를 통해 최근 90일에 3GW 이상을 추가했고 2029년 10GW 목표를 조기 초과할 수 있다고 밝혔습니다. 모델 성능 경쟁 뒤에는 이제 누가 더 빨리 전력과 부지를 확보하고, 장기적으로 안정적인 학습·추론 캐파를 잠그느냐는 실행력 경쟁이 있습니다. ...
