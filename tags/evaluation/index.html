AI/ML 트렌드: 에이전트는 더 물리적이고, 더 위험하고, 더 운영형이 되고 있다

오늘 #ai-ml-trends에서 가장 눈에 띈 흐름은 에이전트 AI가 챗봇이나 코드 생성기를 넘어 물리 세계, 장기 실행 워크플로, 보안 운영으로 빠르게 내려오고 있다는 점입니다. Meta의 humanoid AI 강화를 위한 robotics 스타트업 인수, NVIDIA의 DeepSeek V4 배포 가이드, Crab 같은 agent sandbox checkpoint 연구, Exploration Hacking과 FlashRT 같은 post-training·red-teaming 연구가 모두 같은 방향을 가리킵니다. 이제 중요한 질문은 “모델이 똑똑한가?”보다 “이 시스템을 오래, 싸게, 안전하게 굴릴 수 있는가?”에 가까워졌습니다. 첫 번째 축은 physical AI입니다. Meta가 humanoid AI 역량을 위해 로봇 AI 회사를 인수한 것은 로봇이 더 이상 별도 산업의 실험실 프로젝트만은 아니라는 신호입니다. VLM, long-context agent, world model, reinforcement learning이 서로 붙으면서 AI는 화면 안의 조언자가 아니라 센서와 액추에이터를 가진 작업자로 확장되고 있습니다. 이는 모델 경쟁의 기준도 바꿉니다. 자연어 답변 품질만으로는 부족하고, 실제 환경에서 실패를 감지하고, 복구하고, 안전 경계를 지키는 능력이 중요해집니다. ...

AI/ML 트렌드: AI 경쟁은 이제 풀스택 인프라와 거버넌스 시장으로 이동하고 있다

오늘 #ai-ml-trends 채널에서 가장 크게 보인 흐름은 AI 경쟁이 더 이상 “좋은 모델 하나”의 문제가 아니라는 점입니다. Amazon AWS의 AI 수요 성장, Alphabet Cloud의 TPU 직접 판매, OpenAI의 Stargate 컴퓨트 인프라 계획, Nvidia B300 서버의 중국 가격 급등은 모두 같은 방향을 가리킵니다. AI는 모델 API 시장을 넘어 전력, 칩, 클라우드, 데이터센터, 배포 파트너까지 묶인 풀스택 인프라 산업이 되고 있습니다. 특히 Google Cloud와 OpenAI의 움직임은 상징적입니다. Google은 클라우드 성장의 핵심 동력으로 AI를 전면에 내세우는 동시에 TPU를 고객에게 직접 판매하기 시작했습니다. 이는 hyperscaler가 단순히 모델을 빌려주는 단계를 넘어, 자체 칩과 클라우드, 엔터프라이즈 배포를 한 번에 묶어 팔겠다는 신호입니다. OpenAI도 Stargate 업데이트를 통해 최근 90일에 3GW 이상을 추가했고 2029년 10GW 목표를 조기 초과할 수 있다고 밝혔습니다. 모델 성능 경쟁 뒤에는 이제 누가 더 빨리 전력과 부지를 확보하고, 장기적으로 안정적인 학습·추론 캐파를 잠그느냐는 실행력 경쟁이 있습니다. ...
