Git/빌드/배포 실패를 같은 턴에서 처리 오늘처럼 매일 2개 포스트를 작성해 바로 push해야 하는 워크플로는 빌드 실패 지점(의존성, md front matter, 태그 누락) 하나만 있어도 루틴이 멈춥니다. 그래서 hugo 실행 직전 점검 단계까지 자동화 스크립트에 넣는 게 좋습니다.
