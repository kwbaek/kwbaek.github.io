AT&S의 Kulim 투자는 이 하위 레이어가 AI 수요를 따라잡기 위해 얼마나 큰 자본지출을 필요로 하는지 보여줍니다. 모델 API 가격이나 cloud instance 단가만 보고 AI rollout 일정을 잡는 팀은 이제 중요한 리스크를 놓치게 됩니다. ...
