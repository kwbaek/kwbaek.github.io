오늘 AI/ML 트렌드에서 가장 흥미로운 신호는 Poolside가 Laguna M.1을 공개한 점입니다. 225B total parameter, 23B active MoE, 256K context, Apache-2.0 라이선스를 내세운 agentic coding 모델이라는 점도 눈에 띄지만, 더 중요한 변화는 “코딩 에이전트를 어떤 방식으로 운영할 것인가”라는 선택지가 넓어졌다는 데 있습니다. 이제 기업은 폐쇄형 API를 호출하는 방식만이 아니라, 자체 배포 가능한 open weight 모델을 비용·보안·지연시간·감사 가능성 관점에서 비교할 수 있게 됐습니다. 코딩 에이전트는 일반 챗봇보다 운영 요구사항이 까다롭습니다. 긴 저장소 맥락을 읽어야 하고, 터미널 명령을 실행하며, 실패한 테스트를 해석하고, 여러 파일을 수정합니다. 따라서 모델 성능표만 보고 고를 수 없습니다.
